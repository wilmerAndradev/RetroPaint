# 🎨 RetroPaint

🕹️ App de dibujo y pintura retro para el navegador — capas, filtros, pixel art, selector de color avanzado y más. Sin backend, sin instalación.

Aplicación de dibujo y pintura web con estética retro. Inspirada en los clásicos programas de pintura pero con funcionalidades modernas: capas, filtros, modos de fusión, selector de color avanzado, atajos de teclado y más. Corre 100% en el navegador sin backend.

## Stack
- HTML5 Canvas
- CSS3 + Variables CSS
- JavaScript Vanilla
