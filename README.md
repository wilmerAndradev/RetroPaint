# 🎨 RetroPaint

🕹️ Aplicación de dibujo y pintura retro para el navegador — capas, filtros, pixel art, selector de color avanzado y más. Sin backend, sin instalación.

## Stack

### Core
- **React 19** — UI framework con hooks
- **TypeScript** — tipado estático
- **HTML5 Canvas** — motor de dibujo
- **Vite** — bundler y servidor de desarrollo

### Estado & lógica
- **Zustand 5** — gestión de estado global
- **Immer** — mutaciones inmutables del estado

### Estilos
- **Tailwind CSS 4** — utilidades CSS
- **CSS3 + Variables CSS** — temas y estilos retro

### Animaciones
- **GSAP 3** + **@gsap/react** — animaciones de interfaz

### Calidad de código
- **Biome** — linter y formatter
- **ESLint** — reglas adicionales para React
- **Vitest** — pruebas unitarias
